Display Block Flow - Não é possível usar um block flow pois ele não existe, o block só existe no próprio display e o flow unicamente para o flow-root

Display Block Flex - O block flex não existe, o block pertence a sua classe de display e o flex a dele.

Display Block Grid - Um block grid não pode existir, já que o block sempre vai criar um bloco naquela linha e pular para a próxima e o grid cria uma grade, coisas diferentes.

Display Block Flow-Root - é uma sintaxe CSS que cria um elemento de bloco que estabelece um novo contexto de formatação de bloco (BFC), contendo completamente elementos flutuados.
